**Fitur-fitur:**

>1. Mengganti Username, Bio/Tentang, Foto Profil
>2. Chat Pribadi Antar Pengguna
>3. Grup Chat
>4. Global Chat
>5. Feed/Postingan/Story
>6. Pesan suara dan gambar pada chat

>8. Background Chat
>9. Tema (Light/Dark)
>10. Menghapus Akun
